
Hide Language Links: README.txt
=========================

This module provides functionality to hide language links in the language switcher and content translationlinks for certain roles.
Please note that this module does not provide any access restrictions for certain languages, it only hides links in the language switcher provided by the core locale module and content translation links provided by the core translation module.

Dependencies:
- Locale, drupal core
